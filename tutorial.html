Folder structure

Each template folder contains exactly three files:

  • index.html — markup with inline SVG logo and favicon (data URI)
  • style.css — all styles and background animations
  • script.js — countdown controls and interactions

Change logo (inline SVG)

Open index.html and find the <svg class="logo-svg" ...>. Replace the text elements or shapes. You can paste your own SVG; keep width/height reasonable (e.g., height ≈ 44px).

To edit brand text, update the brand text inside the SVG group.

Use a custom favicon (SVG)

Find the favicon link in <head>:

<link rel="icon" type="image/svg+xml" href="data:image/svg+xml;utf8,<svg ...>...</svg>" />

Replace the inline SVG or point to an external favicon.svg file:

<link rel="icon" type="image/svg+xml" href="favicon.svg" />

Change colors & background

In style.css, adjust the CSS variables under :root:

:root {
  --primary: #7F00FF;
  --accent: #E100FF;
  --bg: #0b0d12;
}

For background animations, tweak the .decor rules per variant class on the <body>.

Change title, paragraph, icons

Edit the .title and .tagline in index.html. Icons come from CDN (Boxicons and Remix Icon). Swap classes like bx bx-time-five or ri-home-5-line to change icons.

Countdown counter

In script.js:

  • Change duration: set let seconds = 10; to your preferred number.
  • Reset: the reset() function sets it back to 10. Modify there.
  • Redirect: uncomment location.href = '/' and change the path.

Reset styles quickly

Delete or adjust any variant class on <body class="variant-..."> to switch background styling.

Performance tips

  • Prefer SVGs over heavy images.
  • Keep gradients and blurs reasonable for low-end devices.
  • Host icon CSS on your own CDN for production if needed.
